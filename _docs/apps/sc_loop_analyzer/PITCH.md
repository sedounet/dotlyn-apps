# SC Loop Analyzer — PITCH.md

## Concept
Application desktop Flutter pour Windows permettant d’analyser et chronométrer les boucles de travail (loops) dans Star Citizen, avec gestion de profils personnalisés et historique des sessions.

## Identité visuelle
- Styleguide Dotlyn (orange E36C2D, gris 2C2C2C)
- Typo Satoshi (titres), Manrope (UI)
- Icônes Remix Icon

## Public cible
- Joueurs Star Citizen cherchant à optimiser leurs boucles de farm/trade
- Streamers, orga, theorycrafters

## Différenciation
- Ultra-rapide, UX minimaliste
- Profils personnalisables
- Statistiques et export simples

## Métriques succès
- Nombre de sessions trackées
- Nombre de profils créés
- Feedback utilisateurs
