# Objectif
Créer une app Flutter Windows "SC Loop Analyzer" pour chronométrer et analyser les boucles de travail Star Citizen.

## Tâches principales
- Générer structure dossier app + doc
- Générer UI de base (profils, session, historique, stats)
- Mettre en place modèles, services, providers
- Stockage SQLite (profils, sessions)
- Respecter conventions Dotlyn Apps

## Contexte technique
- Flutter Windows
- sqflite_common_ffi, Provider, intl, SharedPreferences
- UI Material 3, responsive, styleguide Dotlyn

## Critères de succès
- App fonctionnelle, légère, extensible
- CRUD profils, suivi session, historique, stats
- Code commenté, clair, prêt à étendre
