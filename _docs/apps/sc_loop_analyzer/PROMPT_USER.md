Créer une application Flutter Windows pour chronométrer et analyser les boucles de travail dans Star Citizen, avec gestion de profils, stockage SQLite, et interface simple.

Fonctionnalités principales :
- CRUD profils (nom, étapes, unité, notes)
- Suivi session (étapes, quantités, commentaires, calculs de durée)
- Historique sessions (tableau, recherche, export CSV)
- Statistiques par profil

Stack : sqflite_common_ffi, Provider, intl, Material 3, responsive, SharedPreferences
